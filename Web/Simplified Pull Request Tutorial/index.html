How to Make a Pull Request

1. Find an interesting repository that you would like to make changes to.

2. Then, you fork the repository to your Github account.

3. Clone the forked repository to your local machine.

4. Create a new branch for the changes that you want to make.

5. Make the changes on the new branch and publish those changes to Github.

6. Create pull request on Github and wait for owner of repo to pull in your changes.
